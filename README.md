# Pretreatment Multiparametric MRI Habitat Mapping Predicts Early Tumor Regression and Prognosis in Nasopharyngeal Carcinoma

## Overview

This repository contains the official implementation for the paper:

**Pretreatment Multiparametric MRI Habitat Mapping Predicts Early Tumor Regression and Prognosis in Nasopharyngeal Carcinoma**

The code supports the complete pipeline for habitat-based radiomics analysis.



## Requirements

- Python ≥ 3.8
- NumPy
- SciPy
- SimpleITK
- PyRadiomics
- scikit-learn
- pandas
- matplotlib
- nibabel
- PyTorch ≥ 1.10 (for deep learning)
- openpyxl (for Excel file handling)
